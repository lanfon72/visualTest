Data Visualization

NVD3 Test Project

What is data visualization?

資料視覺化早在電腦發展的早期就已經被提出，相關資料可參考維基百科。 在這主要是使用 D3.js library 及其衍生 library: NVD3 ，利用 html5 所提供的新標籤，產生視覺化圖表(順便練習一下CSS3跟javascript)。

關於資料

資料的部份，因為之前蜘蛛人Ronnywang實況了用php爬監察院資料，所以就順便用php寫了一支crawler去爬pad的wiki(source code也在github)。政治獻金的部份可參考：蘋果新聞、FB粉絲頁。

關於圖表

圖表的部份，利用爬下來的資料測了可排序表格(sortedtable)、圓餅圖(piechart)和散佈圖(scatterChart)。相關的圖表其實並不是什麼太新的技術，利用原始資料(rawdata)也可以在 Excel 上進行操作，但 D3js 之所以重要的原因在於：你不用裝 excel 就可以看了啊笨蛋=_=(而且還跨平台)。
